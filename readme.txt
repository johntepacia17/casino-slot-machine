PIXIJ S SLOT MACHINE (5x3) — LOCAL SETUP GUIDE
============================================


------------------------------------------------
PREREQUISITES
------------------------------------------------
You need:

- Node.js 18+ (Node 20 LTS recommended)
- npm (included with Node)

Check installation:
node -v
npm -v


------------------------------------------------
1) CLONE THE REPOSITORY
------------------------------------------------
git clone https://github.com/johntepacia17/casino-slot-machine.git
cd casino-game


------------------------------------------------
2) INSTALL DEPENDENCIES
------------------------------------------------
Using npm:
npm install

Or with yarn:
yarn

Or with pnpm:
pnpm install


------------------------------------------------
3) RUN THE PROJECT (DEVELOPMENT MODE)
------------------------------------------------
npm run dev

The terminal will show a local URL, usually:
http://localhost:5173

Open that URL in your browser.


------------------------------------------------
CONTROLS
------------------------------------------------
- Click SPIN           → normal spin
- Hold SPACE           → turbo spin (fast continuous spins)
- Click + / -          → increase or decrease bet

------------------------------------------------
TECH STACK
------------------------------------------------
- Vite
- PixiJS

END
